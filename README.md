Source
=====================
Source is run on the Ionic Framework. 


## Setting up the Prokect

First Install Ionic globally

```
$ npm install -g ionic
```

Install Node dependancies
```
npm install
```

Install gulp
```
gulp install
```



## Running the app

To launch the App in a browser

```
$ ionic serve
```

For Live reload
```
gulp
```
This will watch the sass files for any changes
